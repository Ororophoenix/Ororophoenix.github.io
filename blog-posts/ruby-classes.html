Ruby Classes

Not classes like boring lectures; More like OCD organization.

Hey yall we are back at this again. So this week we are talking about classes in ruby. So lets just clear the air. Classes are essentially a nice way of organizing your code and making it more friendly. So without classes you have all of these different methods where you cant use them in other methods. Think of it like cliques in high school. Without classes each method is a different clique like the plastics and pretty much tell the others that you can sit with us because on Wednesdays we wear pink.

Classes breaks down all of the little cliques and essentially works like a OCD persons dream.
